# Operational Audit Framework: The "Friction-to-ROI" Assessment

*This is the core discovery tool used by Syna Systems to identify operational bottlenecks. It is designed to be engineering-grounded and implementation-oriented.*

## 1. The "Human Middleware" Audit
**Goal:** Identify where humans are acting as API connectors.
- **Data Transfers:** Which tasks involve copying/pasting between tools?
- **Status Updates:** How much time is spent "checking in" on task progress?
- **Manual Filtering:** Is a human reading 100 things to find the 5 that matter?

## 2. The Scaling Constraint Analysis
**Goal:** Find the "One Person" bottleneck.
- **Approval Chains:** Where does a process stop for a single signature/check?
- **Tribal Knowledge:** Which tasks can only be done by the founder/senior lead?
- **Cognitive Load:** Which team is "too busy" to optimize their own workflow?

## 3. Automation ROI Matrix (The "Immediate Yes")
**Goal:** Pick the lowest hanging fruit for a 14-day Pilot.
- **Complexity (Low):** No fuzzy logic, just if-this-then-that.
- **Frequency (High):** Happens 10+ times a day.
- **Business Impact:** Saves at least 5 hours/week per person.

## 4. Security & Compliance Check
- **PII Handling:** Are we touching customer data?
- **System Access:** Do we have API keys or do we need OAuth?
- **Audit Trails:** How will we prove the automation worked correctly?
